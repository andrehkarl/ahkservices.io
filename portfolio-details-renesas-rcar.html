CASE STUDY

Optimizing Performance and Real-Time Capabilities for Next-Generation Platforms

At AHK Services, we empower clients to achieve peak performance and real-time precision in their embedded systems. Here's how we helped optimize a crucial performance measurement tool for one of Renesas' key automotive customers using the market leading VH4 RCar platform:

Strategic Performance Tool Migration Zephyr: We meticulously analyzed an existing performance measurement tool and its driver, originally running on embedded Linux on a Cortex A76 (V4H platform). Requirement was to port this critical tool to the Real-Time Operating System Zephyr RTOS running on a Cortex R52 on the very same platform, unlocking new levels of responsiveness and efficiency.

Trace data importer: Our experts successfully ported the performance measurement tool which also involved designing and implementing new tools enabling retrieval of data captures from Arm's CoreSight Performance Monitoring Units (PMUs) and Embedded Trace Macrocell (ETM) of the multiple V4H processor cores into the Renesas e2studios trace data plotter and analyser.

Automated Validation for Flawless Operation: To ensure absolute confidence in the new implementation, we designed and implemented a fully automated test suite. This rigorous testing verifies the correct and consistent operation of the ported performance measurement tool, guaranteeing its accuracy and reliability in the new RTOS environment.
