CASE STUDY

Research project for HbbtV-TA Ad Monetization for Samsung

We led the research, requirements analysis, architecture, and implementation of a prototype system.

This involved establishing what Ad impression data and user interaction metrics can be retrieved from the Smart TV platform and identifying innovative ways of correlating and collecting these metrics.
The solution included necessary adaptations to the HbbTV application runtime and the implementation of a sophisticated external database server backend.

As part of the work a comprehensive system and defect analysis of the HbbTV subsystem has been performed to ensure robust functionality of the platform.
We enhanced software reliability by expertly retrofitting unit tests using Google Test and Google Mock, significantly boosting test coverage enabling safe regression free development cycles.
This backend now gathers targeted advertisement impression metrics, forming the basis for seamless and compliant implementation of the HbbTV-TA standard for advanced Ad monetization on Smart TV products.
